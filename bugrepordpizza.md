# Баг-репост пицца дня со скидкой 10%

- ID - **bug01**
- Заголовок - **Пицца со скидкой 10 % в корзине без скидки**
- Описание - **Пицца со скидкой 10% при добавлении в корзину должна отображаться в корзине с 10% скидкой**
  - Предусловия - **Нажать "*Добавить*" на пиццу со скидкой 10%**
  - Тестовые данные - **https://slqamsk.github.io/cases/pizza/ Версия без ошибок**
  - Шаги воспроизведения - **Нажать добавить, нажать корзина**
  - Ожидаемый результат - **При добавлении в корзину должна отображаться ссума со скидкой 10%**
  - Наблюдаемый результат - **При добавлении в корзину пицца отображаться без скидки 10%**
- Найден в версиях - **https://slqamsk.github.io/cases/pizza/ Версия без ошибок**
- Критичность - **Высокая**
- Приоритет - 1 место
